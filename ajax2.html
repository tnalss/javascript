$.ajax() 메소드 실습 (p.499)

서버로 비동기 ajax 요청을 하며, 주로 로그인 처리/ 회원가입 처리/글쓰기 처리(=게시판,블로그) 등과 같이 DB와 연동 후 처리 결과만을 반환하는 경우에 사용하면 좋다.

$.get(), $.post() 메소드 대신 이 메소드 $.ajax()를 사용해서 Ajax를 구현할 수 있다.

참고로 jQuery의 Ajax 통신은 jQuery 고유의 방식이 아닌 Ajax 라는 javascript 통신방식을 의미하고 , 이때 XMLHttpRequest라는 메소드를 이용해 처리르 손쉽게 대신해준다.

$.ajax() 메소드 형식

$.ajax({type: value , url : value, data: value, success:function(data){...}})
  • type:서버에 요청할 HTTP방식(get or post) 지정-get이 기본값
  • url:서버에 요청할 url, 필수요소
  • data : 전송할 데이터로 문자열 또는 객체로 기술한다. 선택요소
  • success : 서버가 처리한 요청이 성공하면 function(data){성공하면 실행할 코드} 콜백함수가 실행된다. 매개변수 data는 처리결과를 반환받는다. 이 처리 결과에 따라 작업수행이 필요한 경우 사용한다.
